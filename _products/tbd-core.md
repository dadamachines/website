The tbd-core is the core board from the [tbd-16](/shop/tbd-16/), available separately for instrument designers and product developers who want to build custom hardware around the TBD platform.

Same three processors, same 50+ DSP plugins, same firmware. All audio, MIDI, and USB connectors come assembled on the board. Connect your custom buttons, encoders, LEDs, and displays via the 30-pin FFC connector — design your own enclosure and controls.

Fully open source (GPL 3.0). The same web interface works out of the box — no software changes needed.
