The tbd-16 is a complete desktop instrument. Plug in USB-C power, connect speakers or headphones, and it boots straight into the default Groovebox / Drum Machine firmware with 16 tracks. No computer, no Eurorack case, no setup required.

50+ DSP plugins — synths, effects, drum machines, granular engines — all running at 32-bit float precision with sub-millisecond latency. Three processors (ESP32-P4, RP2350B, ESP32-C6), WiFi 6, Ableton Link, 2× TRS MIDI In/Out, USB MIDI, and a browser-based web interface.

The DSP engine, firmware, and plugin system are fully open source (GPL 3.0), built on [CTAG TBD](https://github.com/ctag-fh-kiel/ctag-tbd). Write your own plugins in C++, build custom MIDI controllers on the RP2350, or use the [tbd-core](/shop/tbd-core/) to build your own product around the same platform.
